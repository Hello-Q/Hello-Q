06.Python-函数
基础 python 函数
Python学习笔记

函数是组织好的，可重复使用的，用来实现单一，或相关联功能的代码段。
函数能提高应用的模块性，和代码的重复利用率。

定义一个函数

  • 函数代码块由def关键字开头，后面接函数的名称和小括号 ()
  • 任何传入参数和自变量必须放在圆括号中间。圆括号之间可以用于定义参数。
  • 函数的第一行语句可以选择性地使用文档字符串—用于存放函数说明。
  • 函数内容以冒号起始，并且缩进
  • return [表达式] 结束函数，选择性地返回一个值给调用方。不带表达式的return相当于返回 None。

基本语法

def functionname (parameters):
    "此处放函数的文档说明"
    function_suite
    return [expression]

实例

def print_str(str):
    "将传入的文本打印出来"
    print(str)
    return

函数的调用

直接输入函数的名字加上括号即可调用，括号中是否由内容根据实际情况

调用函数实例

print_str("我调用函数")

函数的返回值

  • 调用函数时，函数返回的一个执行结果
  • 使用return返回
  • 可以没有返回值，但建议没有返回值时也写一个return None， 没有返回值的函数默认返回None
  • 函数一旦执行return， 则函数立即结束，后面的代码不再执行

参数的传递

参数负责给函数传递一些必要的数据或者信息

  • 形参：在函数定义的时候要用到的参数，没有具体的值，占位
  • 实参： 在函数调用时，实际传入的值

实参的传递

位置实参

在调用函数时，Python必须将函数调用中的每个实参关联到函数定义中的一个形参，最简单的关联方式是基于实参的顺序，这种关联方式称为位置实参

实例

def test(a, b, c):
    print(a + b)
    return None

test(1, 2, 3)

# 调用函数时将实参1, 2, 3 分别对应到形参a, b, c

位置实参的顺序以及数量很重要，必须一一对应关联

关键字实参

关键子实参是调用函数时传递给函数的名称-值对，直接在实参中将名称和值关联起来了，因此向函数传递参数时不会混淆，关键字实参无需考虑函数调用的实参顺序

实例

test(b=2, a=1, c=3)

使用关键子参数时，一定要准确的指定函数中定义的形参名

关键字参数是调用时的概念

默认值实参

在编写函数时，可以给每个形参指定默认值。在调用函数时给形参提供了实参时，Python将使用提供的实参，否则使用形参的默认值

实例

def test(a, b, c=3):
    print(a+b)
    return None

test(1, 2)
test(1, 2, 4)
# 这两种调用方法都是可以的，第一个未提供c值，pyhton会使用默认值3。第二种提供c=4，pyhton会使用提供的实参4

使用默认值时，在形参中必须先列出没有默认值的形参，再列出有默认值的形参，否则Python无法正确理解位置实参

收集实参

有时候并不知道函数需要接受多少个实参，python允许函数从调用语句中收集任意数量的实参。

实例

# 定义一个函数，打印出班级学生姓名

def stu_name(*names):
    for name in names:
        print(name)

stu_name('zhang', 'liu', 'wang')

形参name是. 中的\会创建一个名为names的空元祖，在调用时，会将所有的值装入元祖。

  • 关键字收集实参

有时候需要接受任意数量的的实参，但预先并不知道传递给函数的是什么信息，可以使用关键字收集实参。

实例

# 收集班级中所有的学生的相关信息
def stu_info(name, age, **info):
    info = {}
    info['name'] = name
    info['age'] = age
    for k, v in info.items():
        info['k'] = v
    return info

stu_info('张三', '18', tel='1302566', hobby='basketball', )

**会收集多余位置实参的信息，并将信息在函数内部组成一个dict，在函数直接将info当成一个dict使用。

以上实参可以混合使用，但在定义函数是必须按照位置实参→收集实参→收集关键字实参的顺序定义，在调用时，对应位置的参数进入位置实参，多余的进入收集实参，键值对进入收集关键字实参。

变量的作用域

作用域是指可用的范围、变量的有效区域

Python的作用域

Python一共有4中作用域

  • L（Local）：最内层，包含局部变量，比如一个函数/方法内部。
  • E（Enclosing）：包含了非局部(non-local)也非全局(non-global)的变量。比如两个嵌套函数，一个函数（或类） A 里面又包含了一个函数 B ，那么对于 B 中的名称来说 A 中的作用域就为 nonlocal。
  • G（Global）：当前脚本的最外层，比如当前模块的全局变量。
  • B（Built-in）： 包含了内建的变量/关键字等。，最后被搜索

Python在作用域查找变量的顺序为L→E→G→B，Python由内 向外查找，直至找到，如查找到B仍然未找到，则抛出未定义异常。

Python 中只有模块（module），类（class）以及函数（def、lambda）才会引入新的作用域，其它的代码块（如 if/elif/else/、try/except、for/while等）是不会引入新的作用域的，也就是说这些语句内定义的变量，外部也可以访问。

全局变量与局部变量

全局变量：定义在函数外部的变量，可在整个程序范围内使用，拥有全局作用域

局部变量：定义在函数内部的变量，只能在定义的函数内部使用，只有局部作用域

c = 0  # 定义在函数外部-全局变量
def sum(a, b):
    c = a + b  # 定义在函数内部-局部变量
    print('局部变量值为：', c)
    return c
sum(1, 2)
print('全局变量值为', c)

>>> 局部变量值为： 3
>>> 全局变量值为 0

global与nonlocal关键字

在全局作用域中无法使用局部变量，局部作用域中可以读取全局变量和修改全局变量中的可变数据类型（列表、集合、字典），不能修改不可变数据类型。

  1. 如果想要在函数内部修改全局变量，需要使用global关键字
c = 2
def change():
    global c  # 此处使用global修饰变量为全局变量
    c += 1
    return
change()
print(c)

>>> 3

如果不使用global关键字，将会报错UnboundLocalError: local variable ‘c’ referenced before assignment

  1. 在嵌套函数中修改外部函数变量需要使用nonlocal关键字
def f():
    c = 2
    def change():
        nonlocal c
        c += 1
    change()
    print(c)
f()

>>> 3

如不使用nonlocal关键字，将会报错UnboundLocalError: local variable ‘c’ referenced before assignment

此时也不能使用global关键字，因为在全局变量中没有c这个变量，所以会报未定义错误。

globals()与locals()函数

  • globals()函数可以获取当前作用域全局变量
  • locals()函数可以获取当前作用域的局部变量

递归函数

递归函数的定义

递归函数就是定义一个函数，然后再函数内部自己调用自己的函数

示例

def f(x):
    print(x)  # 计算时打印
    if x > 0:  # 递归中要有结束条件，否则程序将一直运行下去
        f(x - 1)  # 在函数内部调用自己，参数改变
    print(x)  # 返回打印


f(3)

>>> 3
>>> 2
>>> 1
>>> 0
>>> 0
>>> 1
>>> 2
>>> 3

递归函数会一层一层进行计算，最后在递归处一层一层将计算结果返回。

递归函数需要一层一层计算，效率较低，一般递归能实现的for循环也可以，所以除特殊场景，一般不是用递归。

递归实现阶乘

def f(x):
    if x == 1:
        return 1
    else:
        return x*f(x-1)  #调用自己

res = f(3)
print(res)

>>> 6

计算3的阶乘，程序运行

  1. 3带入后进入else计算，函数返回3*f(3-1)，即f(3)=3*f(2)
  2. 此时需要计算出f(2)是什么，继续将2带入函数计算，此时函数返回2*f(2-1), 即f(2)=2*f(1)
  3. 将1带入函数，此时x==1，程序返回1，即f(1) == 1
  4. 得知f(1)==1后，程序开始返回结果，f(2)=2*1=2，对应第2步，计算得出f(2) = 2
  5. 计算得出f(2) = 2，也就是f(3)=3*f(2)=3*2=6
f(3) =  # 将3带入后，程序返回f(3)=3*f(2)
    3*f(2)  # 此时程序需要计算f(2)是什么，函数返回f(2)=2*f(1)
        2*f(1)  # 此时需要计算f(1)是什么，函数返回f(1)=1
            f(1) ==> 1  # 得出结果，f(1) = 1
        2*1  # 对应得出f(2)=2*f(1) = 2*1=3
    3*2  # 对应得出f(3)=3*f(2) = 3*2 = 6
f(3) = 6  # 得出最终结果f(3) = 6

得出结果=6

阶乘的for循环实现

Num = int(input('请输入一个整数： '))

for i in range(1,Num):
    Num = Num*i
print(Num)

递归函数实现斐波那契数列

斐波那契数列定义：

指的是这样一个数列：1、1、2、3、5、8、13、21、34、……，从第三位开始，每一位都等于前两位的和。

def f(n):
    if n == 1 or n == 2:
        return 1
    else:
        return f(n-1)+f(n-2)

res = f(7)
print(res)

>>> 13
  • 当n=1 或者2时，程序返回1
  • 当n大于2时，返回f(n-1)+f(n-2)
  • 直到程序运行到n=2，返回1
  • 得出结果层层返回

回调函数

回调函数是指在一个函数中传递的参数是另一个函数，并在函数中使用了传递的函数，那个这个被使用的函数称为回调函数

def callback():
    print('这是一个回调函数')

def test(f):
    print('主函数')
    f.()   # 调用回调函数
    print('主函数结束')

>>> 主函数
>>> 这是一个回调函数
>>> 主函数结束

闭包函数

定义一个函数，在函数的内部再定义一个内嵌函数,内嵌函数使用了外函数的变量，并且外函数的返回值为内部定义的内嵌函数，则这个函数为闭包函数

示例

def a():
    x = 1
    def b():
        nonlocal x
           x += 1
        print(x)
    return b
res = a()
res()
res()
res()

>>> 2
>>> 3
>>> 4

闭包函数的作用

  • 保护变量
  • 保存函数的状态信息，使函数的局部变量信息依然可以保存下来

示例中，虽然a函数的调用已经结束，但x的值仍然保持状态。所以闭包函数会比较占用资源，当变量不使用时，应及时删除。

闭包函数的检测

使用调用外函数名.__closure__检测，如果是闭包函数，返回一个cell，否则返回None

print(a().__closure__)

>>> (<cell at 0x0000012F0AB05F18: int object at 0x00007FF983BED420>,)

匿名函数

Python中使用lambda定义，匿名函数不使用def定义，没有名字所以称为匿名函数，一般一些逻辑简单的函数使用。

# 实现两个数字相加
res = lambda x, y : x+y
print(res(4, 5))

>>> 9

配合map使用将一个列表中的数据统一进行相同的计算

res = list(map(lambda x:x*3, [1, 3, 5]))  # 对列表中所有的元素乘以3
print(res)

>>> [3, 9, 15]

内置函数

map()

语法：

  • map(function, iterable, ...)
  • function – 函数
  • iterable – 一个或多个序列

功能：将序列中的每个元素依次放入函数中，然后将函数的返回值组成新的迭代器

l1 = [1, 2, 3, 4]
l2 = [5, 6, 7, 8] 

def f1(x, y):  # 将两个元素相乘
    return x * y


res = map(f1, l1, l2)  # 将l1和l2中的每个元色依次相乘
print(list(res))

>>> [5, 12, 21, 32]

def f2(x):  # 将输入的参数乘10
    return x * 10

res = map(f2, l1)  # 将l1中的每个参数乘以10 
print(list(res))

>>> [10, 20, 30, 40]

res = map(lambda x: x*10, l1)  # 配合lambda使用
print(list(res))

>>> [10, 20, 30, 40]

reduce()

语法

  • reduce(function, iterable[, initializer])
  • function – 函数，这个函数要有两个参数
  • iterable – 可迭代对象
  • initializer – 可选，初始参数

功能：将一个数据集合中所有数据中的第1、2个元素传入function中进行计算，并将函数返回的结果和第三个元素再次传入function中计算，以此类推，直至最后一个元素参与计算。

使用reduce()需要在functools中导入

from functools improt reduce
def add(x, y) :            # 两数相加
     return x + y

res = reduce(add, [1,2,3,4,5])   # 计算列表和：1+2+3+4+5
print(res)
>>> 15
res = reduce(lambda x, y: x+y, [1,2,3,4,5])  # 使用 lambda 匿名函数
print(res)
>>> 15


def str_(x, y):  # 将一个可迭代对象中所有的元素连接成一个字符串
    return str(x) + str(y)


res = reduce(str_, [1, '2', '3', 4, 5])
print(res)
>>> 12345

filter()

语法：

  • filter(function, iterable)

  • function：判断函数

  • iterable：可迭代对象

功能：将可迭代对象中的元素依次带入判断函数进行判断，若函数返回True，则该元素被保留，放入新的迭代器中，filter()函数返回新的迭代器。

def odd(x):  # 判断一个数字是不是奇数
    if x % 2 == 0:
        return True
    else:
        return False


ls = [1, 2, 3, 4, 5, 6, 7]
res = filter(odd, ls)
print(list(res))
>>> [2, 4, 6]
res = filter(lambda x: True if x % 2 == 0 else False, ls)  # 使用匿名函数
print(list(res))
>>> [2, 4, 6]

sorted()

语法：

  • sorted(iterable, key=None, reverse=False)
  • iterable：一个可迭代对象
  • key：处理函数
  • reverse：是否对处理结果反序

功能：将一个可迭代对象放入key函数中，sorted()函数会根据key函数返回值对可迭代对象进行排序组成新的迭代器。

ls = [1, -2, 3, -4]
res = sorted(ls)  # 对ls列表进行排序
print(res)
>>> [-4, -2, 1, 3]

res = sorted(ls, reverse=True)  # 对结果反序
print(res)
>>> [3, 1, -2, -4]

res = sorted(ls, key=abs)  # 对ls求绝对值后排序
print(res)
>>> [1, -2, 3, -4]

该函数根据key函数对元素进行处理后进行排序，但返回的是函数处理前的元素组成的迭代器。

zip()

语法：

  • zip(*iterables)
  • iterables:可迭代对象

功能：将多个可迭代对象中每个迭代器的第i个元素组合成一个元组的迭代器，当输入可迭代对象中的最短的一个被耗尽时，迭代器停止迭代。

ls1 = [1, 2, 3, 4]
ls2 = ['A', 'B', 'C']
ls3 = ('a', 'b', 'c', 'd')
res = zip(ls1, ls2)
print(res, tuple(res))
>>> <zip object at 0x000002159758D588> ((1, 'A'), (2, 'B'), (3, 'C'))

使用*zip()还可以解包一个列表


ls4 , ls5, ls6 = zip(*zip(ls1, ls2, ls3))
print(ls4, ls5, ls6)
>>> (1, 2, 3) ('A', 'B', 'C') ('a', 'b', 'c')
