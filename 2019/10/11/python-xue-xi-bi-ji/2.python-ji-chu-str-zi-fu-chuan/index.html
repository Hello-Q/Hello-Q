字符串是一系列的字符，在Python中，用引号括起来的是字符串，可以使单引号，也可以是双引号。三引号可以表示多行

字符串的表示

name =  'zhangsan'
name = "zhangsan"
name = """
zhang
san
"""

转义字符

  • 用一个特色的方法表示出一系列不方便写出的内容，例如回车键、换行符、退格键
  • 借助反斜杠字符，接上一个或者几个字符，此时这些字符表示的不再是原来的意思，进行了转移
  • 不同系统对换行符由不同的表示
    • Windows： \n
    • Linux：\r\n

实例

#  转义单引号
s = 'Let\'s Go'
print(s)
>>>Let's Go

# 换行转移
s = 'hello \n word'
print(s)
>>> hello 
 word

# 路径的表示
s = 'c:\\user\\abv'
print(s)
>>> c:\user\abv

在Python中，代码末尾单个单斜杠结束代表这行代码未结束，换行继续写。

常用转义字符

转义字符 含义
\ (在行尾时) 续行符
\\ 反斜杠符号
' 单引号
" 双引号
\a 响铃
\b 退格(Backspace)
\e 转义
\000 空
\n 换行
\v 纵向制表符
\t 横向制表符
\r 回车
\f 换页
\oyy 八进制数，yy代表的字符，例如：\o12代表换行
\xyy 十六进制数，yy代表的字符，例如：\x0a代表换行
\other 其它的字符以普通格式输出

格式化

  • 把字符按照一定的格式进行打印或者填充

传统格式化 %

  • 使用％进行格式化

占位符列表

符号 含义
%s 字符串
%d 十进制整数
%f, %F 浮点数十进制形式

占位符的使用

# %s表示简单的字符串
s = "My name is %s"
print(s%小明)
>>> My name is 小明

占位符一般只能被同类型替换，或者替换类型能被转换成占位符的类型

多个占位符的使用，只需要用括号括起来，依次填入

s = "I am %fKG weight, %fm Heigh."
print(s%(60.4, 175.2))
>>> I am 60.400000 weeightt, 175.200000m Heigh.

# 可以指定浮点数小数位数
s = "I am %.2fKG weight, %.2fm Heigh."
print(s%(60.4, 175.2))
>>> I am 60.40 weeightt, 175.20m Heigh.

format格式化

  • 使用函数形式进行格式化，代替%
# 方式1
s = "{} {}!"
print(s.format("Hello", "world"))
>>> Hello, world

# 方式2
s = "{} {}!".format("Hello", "world")
print(s)
>>> Hello, world

 # 方式3 指定参数位置
s = "{1} {0}!".format("world", "Hello")
print(s)
>>> Hello, world

# 参数不足时也可按指定的索引填充，但索引值不能超出提供数量。
s = "I love {0} and {0} love me!".format("小明")
print(s)
>>> I love 小明 and 小明 love me!

# 使用命名参数
s = "我们是{school_name}，我们的网址是{url}。
s = s.format(school_name="猪小萌", url="www.zxmedu.com")
print(s)
>>> 我们是猪小萌，我们的网址是www.zxmedu.com

# 使用字典，需要解包
s = "我们是{school_name}，我们的网址是{url}。

s = "I am {:.2f} m height"
s = s.format(1.7)

str 内置函数

查找类函数

find

查找字符串中是否包含一个子串

语法：S.find(sub[, start[, end]]) -> int

s = "My name is xiaoming"
s1 = "xiaoming"

# 返回第一次发现这个字符串的位置索引，第二次的处理。
print(s.find(s1))
>>> 11

s2 = "xiaohong"
# 未找到匹配项时返回-1
print(s.find(s2))
>>> -1

# 在指定的区间查找
print(s.find(s1, 2, 15))
>>> 11

lfind、rfind 分别代表从左开始查找与从右开始查找。

index

语法：S.index(sub[, start[, end]]) -> int

用法与find相同，区别是find未找到时会返回-1，index会报ValueError

判断类函数

此类函数一般都是使用is开头

isalpha()

判断一个字符串是否全部由字母构成

使用注意：

  • 字符串至少包含一个字符，空字符返回false。
  • 汉字被认为是alpha，此函数不能作为区分英文字母还是汉字。
  • 空格不是alpha，包含空格的字符串返回false

isdigit、isnumeric、isdecimal

判断一个字符串是否是数字

  • isdigit: Unicode数字、byte数字（单字节）、全角数字（双字节）、罗马数字
  • isnumeric: Unicode数字、全角数字（双字节）
  • isdecimal: Unicode数字、汉字数字、全角数字（双字节）、罗马数字

避免使用，尽量使用正则表达式

islower

判断一个字符串是否全部为小写

  • 空格不影响判断，全部为小写的字符串中包含空格返回True
  • 中文返回False

内容判断类

startswith/endswith判断一个字符串是否以xxx开头/结尾

  • 语法: str.startswith(prefix[,staart[,end]]) - > bool
  • prefix: 被检查字符串; start: 开始位置; end: 结束位置;
s = "I am xiaoming"
print(s.startswith("I am", 0, 20))
>>> True

操作类函数

strip

主要用于删除字符串两边的空格

  • 语法strip(self, chars=None, /)
  • 可以指定chars 要删除的字符
  • 不是删除一个字符，是整个符合条件的连续的所有字符串
  • 同类型的还有lstrip()、rstrip()
s = "zhang xiaoming"
s = s.strip("zhang")
print(s)
>>>  xiaoming

join

主要用于字符串拼接

  • 需要一个可以迭代的内用作为参数
  • 功能是吧啊可迭代字符串拼接在一起，中间使用调用的字符串拼接
ss = ["zhang", "xiao", "ming"]
s = "-"
print(s.join(ss))
>>> zhang-xiao-ming
