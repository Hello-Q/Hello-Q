字符串是一系列的字符，在Python中，用引号括起来的是字符串，可以使单引号，也可以是双引号。三引号可以表示多行

字符串_官方文档

字符串的表示

  1. 单引号定义 ‘’
  2. 双引号定义 “”
  3. 三引号定义(内容可换行) “”” “”” 或者 ‘’’ ‘’’
  4. 字符串定义时,引号可互相嵌套
name =  'zhangsan'
name = "zhangsan"
name = """
zhang
san
"""

转义字符

\可以当做转移符或者换行符

换行符

当本行代码未完成需要换行时,需要在代码末尾加上\表示本行代码没有完成并换行

转义符

  • 用一个特色的方法表示出一系列不方便写出的内容，例如回车键、换行符、退格键
  • 借助反斜杠字符，接上一个或者几个字符，此时这些字符表示的不再是原来的意思，进行了转义
  • 不同系统对换行符由不同的表示
    • Windows： \n
    • Linux：\r\n

实例

#  转义单引号
s = 'Let\'s Go'
print(s)
>>>Let's Go

# 换行转移
s = 'hello \n word'
print(s)
>>> hello 
 word

# 路径的表示
s = 'c:\\user\\abv'
print(s)
>>> c:\user\abv

在字符串前面加一个r可以关闭转义, s = r’hello world’

常用转义字符

转义字符 含义
\ (在行尾时) 续行符
\\ 反斜杠符号
' 单引号
" 双引号
\a 响铃
\b 退格(Backspace)
\e 转义
\000 空
\n 换行
\v 纵向制表符
\t 横向制表符
\r 回车,光标起始位置
\f 换页
\oyy 八进制数，yy代表的字符，例如：\o12代表换行
\xyy 十六进制数，yy代表的字符，例如：\x0a代表换行
\other 其它的字符以普通格式输出

字符串的操作

字符串的 + 操作

字符串相加可以完成字符串的拼接

s1 = 'hello '
s2 = 'world'

s = s1 + s2
print(s)
>>> hello world

字符串的 * 操作

字符串乘以一个数字可以重复相应的次数

s1 = 'abc'
s2 = s1 * 3

print(s2)
>>> abcabcabc

字符串的切片操作

字符串的切片只能访问,不能修改
切片的语法

str[开始索引:结束索引:步进值]

  • 开始索引默认为0
  • 结束索引默认为最后
  • 步进默认为1

切片示例

s = '床前明月光'

print(s[0])  # 取第一个字
print(s[1])  # 取第二个字
print(s[-1]) # 取最后一个字
print(s[1:3])  # 从第二个字取到第四个字
print(s[0:3:2])  #　从第一个字，每隔一个字去一个字，取到第四个字为止
print(ｓ[::])  # 取整个字符串
print(s[::-1])  # 反着取整个字符串，相当于反序
print(s[::-2])  # 反着每隔一个字取

>>>床
>>>前
>>>光
>>>前明
>>>床明
>>>床前明月光
>>>光月明前床
>>>光明床

格式化

  • 把字符按照一定的格式进行打印或者填充

传统格式化 %

  • 使用％进行格式化

占位符列表

符号 含义
%s 字符串
%d 十进制整数
%f, %F 浮点数十进制形式

占位符的使用

# %s表示简单的字符串
s = "My name is %s"
print(s%小明)
>>> My name is 小明

占位符一般只能被同类型替换，或者替换类型能被转换成占位符的类型

多个占位符的使用，只需要用括号括起来，依次填入

s = "I am %fKG weight, %fm Heigh."
print(s%(60.4, 175.2))
>>> I am 60.400000 weeightt, 175.200000m Heigh.

# 可以指定浮点数小数位数
s = "I am %.2fKG weight, %.2fm Heigh."
print(s%(60.4, 175.2))
>>> I am 60.40 weeightt, 175.20m Heigh.

format格式化

  • 使用函数形式进行格式化，代替%
# 方式1
s = "{} {}!"
print(s.format("Hello", "world"))
>>> Hello world!

# 方式2
s = "{} {}!".format("Hello", "world")
print(s)
>>>  Hello world!

 # 方式3 指定参数位置
s = "{1} {0}!".format("world", "Hello")
print(s)
>>>  Hello world!

# 参数不足时也可按指定的索引填充，但索引值不能超出提供数量。
s = "I love {0} and {0} love me!".format("小明")
print(s)
>>> I love 小明 and 小明 love me!

# 使用关键字参数
s = "我们是{name}，我们的网址是{url}。"
s = s.format(name="百度", url="www.baidu.com")
print(s)
>>> 我们是百度，我们的网址是www.baidu.com。

# 使用容器类型数据传参
s = "我们是{0[0]}，我们的网址是{0[1]}。"
s1 = s.format(['百度', 'www.baidu.com'])
print(s1)
>>>我们是百度，我们的网址是www.baidu.com。


# 使用字典，需要解包
s = "我们是{name}，我们的网址是{url}。"
s_dict = {"name":"百度", "url":"www.baidu.com"}
s = s.format(**s_dict)
print(s)
>>> 我们是百度，我们的网址是www.baidu.com

# 3.7中新增的方法 f

s_dict = {"name":"百度", "url":"www.baidu.com"}
s = f"我们是{s_dict['name']}，我们的网址是{s_dict['url']}。"

print(s)
>>> 我们是百度，我们的网址是www.baidu.com。

# 格式化浮点数指定小数位数。
s = "I am {:.2f} m height"
s = s.format(1.7)
print(s)
>>> I am 1.70 m height

str 内置函数

英文相关函数

capitalize() - 将整个文本首字母大写

返回原字符串的副本，其首个字符大写，其余为小写。

s = 'i love you'
s1 = s.capitalize()
print(s1)
>>> I love you

title - 每个单词首字母大写

返回原字符串的标题版本，其中每个单词第一个字母为大写，其余字母为小写。

>>> 'Hello world'.title()
'Hello World'

upper - 字符串中所有英文字母转大写

返回原字符串的副本，其中所有区分大小写的字符 4 均转换为大写。 请注意如果 s 包含不区分大小写的字符或者如果结果字符的 Unicode 类别不是 “Lu” (Letter, uppercase) 而是 “Lt” (Letter, titlecase) 则 s.upper().isupper() 有可能为 False。

lower - 字符串中所有英文字母转小写

返回原字符串的副本，其所有区分大小写的字符均转换为小写。

swapcase - 大小写互换

返回原字符串的副本，其中大写字符转换为小写，反之亦然。 请注意 s.swapcase().swapcase() == s 并不一定为真值。

检测类函数

此类函数一般都是使用is开头

isalpha() 判断字符串是否全部是字母

使用注意：

  • 字符串至少包含一个字符，空字符返回false。
  • 汉字被认为是alpha，此函数不能作为区分英文字母还是汉字。
  • 空格不是alpha，包含空格的字符串返回false

isupper - 判断一个字符串是否全部为大写

  • 最少有一个字符

islower - 判断一个字符串是否全部为小写

  • 空格不影响判断，全部为小写的字符串中包含空格返回True
  • 中文返回False

istitle - 判断字符串是否符合title要求

  • 要求字符串所有英文单词首字母大写
  • 字符串中有中文不影响判断

isalnum - 判断是否由字符组成

  • 由数字、字母、汉字组成，不能由其他
  • 不能含有空格，否则返回false

isalpha - 是否由中英文组成

  • 不能包含数字
  • 不能包含空格

isspace - 是否全部由空格组成

  • 不能包含除空格之外的其他字符
  • 最少有一个字符

isdigit、isnumeric、isdecimal - 判断一个字符串是否是数字

  • isdigit: Unicode数字、byte数字（单字节）、全角数字（双字节）、罗马数字
  • isnumeric: Unicode数字、全角数字（双字节）
  • isdecimal: Unicode数字、汉字数字、全角数字（双字节）、罗马数字

避免使用，尽量使用正则表达式

startswith/endswith

startswith/endswith判断一个字符串是否以xxx开头/结尾

  • 语法: str.startswith(prefix[,staart[,end]]) - > bool
  • prefix: 被检查字符串; start: 开始位置; end: 结束位置;
s = "I am xiaoming"
print(s.startswith("I am", 0, 20))
>>> True

查找类函数

find - 查找字符串中是否包含一个子串

语法：S.find(sub[, start[, end]]) -> int

s = "My name is xiaoming"
s1 = "xiaoming"

# 返回第一次发现这个字符串的位置索引，第二次的处理。
print(s.find(s1))
>>> 11

s2 = "xiaohong"
# 未找到匹配项时返回-1
print(s.find(s2))
>>> -1

# 在指定的区间查找
print(s.find(s1, 2, 15))
>>> 11

lfind、rfind 分别代表从左开始查找与从右开始查找。

rfind - 从右往左查找字符串是否包含一个子串

语法：S.rfind(sub[, start[, end]]) -> int

  • 与find类似，区别是从右往左查找返回索引值
  • 只有当查找的字符串出现多次，返回结果才会与find有区别

index

语法：S.index(sub[, start[, end]]) -> int

用法与find相同，区别是find未找到时会返回-1，index会报ValueError

rindex - 与rfind类似

查找不到会报错

count - 查找一个字符串在另一个字符串出现的次数

语法： str.count(sub[, start[, end]])
返回子字符串 sub 在 [start, end] 范围内非重叠出现的次数。 可选参数 start 与 end 会被解读为切片表示法。

操作类函数

str.split() - 从左往右 按照制定的字符分割字符串成列表

  • 语法：str.split(sep=None, maxsplit=-1)
  • 返回一个由字符串内单词组成的列表，使用 sep 作为分隔字符串。
  • 如果不指定sep，默认按空格进行拆分。
  • 如果给出了 maxsplit，则最多进行 maxsplit 次拆分（因此，列表最多会有 maxsplit+1 个元素）。
  • 如果 maxsplit 未指定或为 -1，则不限制拆分次数（进行所有可能的拆分）。
>>> s = 'zhang xiao ming'
>>> s.split(' ')
['zhang', 'xiao', 'ming']

str.rsplit - 从右往左 按照指定的字符分割字符串成列表

  • 语法： str.rsplit(sep=None, maxsplit=-1)
  • 返回一个由字符串内单词组成的列表，使用 sep 作为分隔字符串。
  • 如果给出了 maxsplit，则最多进行 maxsplit 次拆分，从 最右边 开始。
  • 如果 sep 未指定或为 None，任何空白字符串都会被作为分隔符。
  • 除了从右边开始拆分，rsplit() 的其他行为都类似于 split()。

str.join - 按指定的字符对可迭代对象进行拼接

  • 语法：str.join(iterable)
  • 需要一个可以迭代的内用作为参数
  • 功能是把可迭代字符串拼接在一起，中间使用调用的字符串拼接
>>> s = ["zhang", "xiao", "ming"]
>>> ss = "-"
>>> print(ss.join(s))
zhang-xiao-ming

str.strip() - 用于删除字符串两边的空格

  • 语法str.strip([chars])
  • 可以指定chars 要删除的字符，不指定默认删除空格。
  • 不是删除一个字符，是整个符合条件的连续的所有字符串
  • 只能处理两侧的字符，中间的不处理。
  • 同类型的还有str.lstrip()、str.rstrip()
  • str.lstrip()只处理左边的（开头）
  • str.rstrip()只处理右边的（结尾）
s = "zhang xiaoming"
s = s.strip("zhang")
print(s)
>>>  xiaoming

str.replace() - 替换字符串中的内容

  • 语法： str.replace(old, new[, count])
  • 返回字符串的副本，其中出现的所有子字符串 old 都将被替换为 new。
  • 如果给出了可选参数 count，则只替换前 count 次出现。
>>> s = 'zhang xiao ming'
>>> s.replace('g', 'a')
'zhana xiao mina'

str.center - 补充指定字符至指定长度

  • 语法： str.center(width[, fillchar])
  • 返回长度为 width 的字符串，原字符串在其正中。
  • 使用指定的 fillchar 填充两边的空位（默认使用 ASCII 空格符）。
  • 如果 width 小于等于 len(s) 则返回原字符串的副本。
  • 默认优先补充左边。
