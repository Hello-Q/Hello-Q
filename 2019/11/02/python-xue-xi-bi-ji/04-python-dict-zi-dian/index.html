python字典的创建

python中，字典用放在花括号中的一系列键-值对表示

info = {"name": "xiaoming", "age": 19}
print(type(info))
>>> <class 'dict'>

字典中的值可以是任何的pyhton中的对象，如数字、字符串、列表乃至字典

使用字典

访问字典的值

要获取字典中的值，可以指定字典名和放在方括号中的键

info = {"name": "xiaoming", "age": 19}
print(info["name"])
>>> xiaoming

添加键-值对

要向字典中添加键-值对，可依次指定字典名、用方括号括起来的键和相关的值

info = {"name": "xiaoming", "age": 19}
info["sex"] = "male"
print(info)
>>> {'name': 'xiaoming', 'age': 19, 'sex': 'male'}

修改字典中的值

要修改字典中的值，可依次指定字典名、用方括号括起来的键以及与该键关联的新值

info = {"name": "xiaoming", "age": 19}
info["age"] = 20
print(info)
>>> {'name': 'xiaoming', 'age': 20}

删除字典中的值

要删除字典中的一个键-值对，可以使用del语句将对应的键-值对直接删除

info =  {'name': 'xiaoming', 'age': 19, 'sex': 'male'}
del info["sex"]
print(info)
>>> {'name': 'xiaoming', 'age': 19}

遍历字典

字典可以用于以各种方式储存信息，以此用多种遍历字典的方式：可以遍历字典的多用键-值对、键或值。

遍历字典的键值对

可以使用for循环来遍历一个字典中的所有信息

info = {'name': 'xiaoming', 'age': 19, 'sex': 'male'}
for k, v in info.items():
    print("key:", k)
    print("value:", v)
>>> key: name
>>> value: xiaoming
>>> key: age
>>> value: 19
>>> key: sex
>>> value: male

申明两个变量k、v储存键-值对中的信息，使用字典名和方法items()将返回一个键-值对列表，使用for循环，依次将键-值对赋值给k、v.

for 循环字典时，键-值对返回的顺序可能与储存顺序不同，python不关系键-值对的储存顺序，只跟踪键和值之间的关联关系

遍历字典的所有键

在不需要字典中的值时，可以使用keys()方法获取字典的所有键。

info = {'name': 'xiaoming', 'age': 19, 'sex': 'male'}
for k in info.keys():
    print("key:", k)
>>> key: name
>>> key: age
>>> key: sex

在遍历字典时，会默认遍历字典的所有的键，以此下列代码以上方效果相同

info = {'name': 'xiaoming', 'age': 19, 'sex': 'male'}
for k in info:
    print("key:", k)
>>> key: name
>>> key: age
>>> key: sex

按照顺序遍历字典的所有键

获取字典的元素时，获取顺序是不可预测的，可以使用sorted()来按照特定的顺序排列列表的副本

info = {'name': 'xiaoming', 'age': 19, 'sex': 'male'}
for k in sorted(info.keys()):
    print("key:", k)
>>> key: age
>>> key: name
>>> key: sex

遍历字典的所有值

当不需要字典中的key时，可以使用方法values()。

info = {'name': 'xiaoming', 'age': 19, 'sex': 'male'}
for v in info.values():
    print("value", v)
>>> value xiaoming
>>> value 19
>>> value male
