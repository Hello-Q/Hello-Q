模块的创建

拓展名为.py的文件中编写函数或其他代码创建模块，这个问价就是模块。

模块中通常定义一些相似功能的类、函数供给其他程序使用。

pizza.py

def make_pizza(size, *toppings):
    """概述要制作的披萨"""
    print("Making a" + str(size) + "-inch pizza with the following toppings:")
    for topping in toppings:
        print("- " + topping)


if __name__ == '__main__':
    print('只有直接运行这个模块时，这部分代码才会被运行')

使用模块可以将函数储存在独立的文件中，隐藏代码的细节，将重点放在程序的高层逻辑上。还可以在众多不同的程序中重用函数。

if __name__ == '__main__': 下的代码，只有在模块直接被运行才有效，被当做模块导入其他的程序中时，这里的代码不会运行。（__name__会输出当前运行的模块名，在自身运行时输出__main__）

包的创建

包就是一个文件夹，里面包含了多个模块

包（文件夹）
|----__init__.py  # 包中都会有一个__init__.py文件
|----pizza.py  # 包中的一个模块
|----a.py  # 包中的一个模块
|----b.py  # 包中的一个模块
|----子包（子文件夹）
|----|----a.py  # a模块
|----|----b.py  # b模块

模块的导入

导入整个模块

使用 import 语句导入模块，需要模块在同级目录或特定的模块目录中。

import pizza

这种导入方法会导入pizza.py 这个模块中所用的函数。

调用函数
module_name.function_name()

import pizza
# 调用函数
pizza.make_pizza(16, 'mushrooms', 'green peppers')
Making a16-inch pizza with the following toppings:
- mushrooms
- green peppers

导入指定的函数

语法：from module_name import function_name

使用逗号分割函数名，可以导入多个函数
from module_name import function_name1, function_name2, function_name3, …

from pizza import make_pizza
make_pizza(16, 'mushrooms', 'green peppers')
>>> Making a16-inch pizza with the following toppings:
>>> - mushrooms
>>> - green peppers

导入所有函数

使用(*)运算符可以导入模块中所用的函数

from pizza import *

# 调用函数
make_pizza(16, 'mushrooms', 'green peppers')

Making a16-inch pizza with the following toppings:
- mushrooms
- green peppers

这样导入的函数，无需使用句点表示法，但是，使用非自己编写的大型模块时，不推荐这样使用，因为导入的模块中，可能模块中的函数名称与项目中相同进而覆盖函数。

as语句

使用as语句可以给导入的函数或者模块起一个别名，方便使用。

给函数指定别名 as

在导入时使用as可以给函数函数指定一个独一无二的别名。解决与现有程序中的名称冲突，或者函数名称太长，起一个简单的外号。

语法： from module_name import function_name as fn

from pizza import make_pizza as mp

# 调用函数
mp(16, 'mushrooms', 'green peppers')
>>> Making a16-inch pizza with the following toppings:
>>> - mushrooms
>>> - green peppers

给模块指定别名 as

也可以使用as 给模块指定别名

语法： import module_name as mn

import pizza as p

# 调用函数
p.make_pizza(16, 'mushrooms', 'green peppers')
>>> Making a16-inch pizza with the following toppings:
>>> - mushrooms
>>> - green peppers

包搜索路径

包的搜索路径可以通过sys模块获取

import sys
print(sys.path)

也可以通过sys.path.append() 添加搜索路径
