python 面向对象 类
Python学习笔记

面向对象编程是最有效的软件编写方法之一，在面向对象编程过程中，编写表示现实世界中的实物个情景的类，并基于这些类创建对象。

创建和使用类

创建一个Dog类

创建一个Dog类，每个实例储存名字和年龄，并且赋予每条小狗蹲下和打滚的能力。

class Dog():   # 1
    """描述小狗的类"""  # 2

    def __init__(self, name, age):  # 3
        """初始化属性name和age"""
        self.name = name  # 4
        self.age = age

    def sit(self):  # 5
        """模拟小狗被命令蹲下"""
        print(self.name.title() + " is now sitting.")

    def roll_over(self):
        """模拟小狗打滚"""
        print(self.name.title() + " rolled_over!")

类的实现

申明一个类

  • 使用class关键字 #1处。
  • #2处为这个类的说明文档，使用三引号括起来。
  • #3处是类的方法，是一个特殊方法，在类被实例化时会自动调用init方法。
  • #4处的变量以self开头，self代表对象本身，以self开头的变量，可供类内所有的方法使用。
  • #5是给类定义的另一个方法。
  • 类由属性和方法（类中的函数称为方法）构成，其他不能出现

    在python2.7中创建类时，需要继承object。

    类的命名

  • 遵循大驼峰命名
  • 避开与系统命名类似

类的实例化

# 变量 = 类名() #实例化了一个对象
dog = Dog("willie", 6)

这样就创建了一个名为“willie”的6岁的小狗。

访问属性

使用句点表示法可以访问实例的属性

dog.name

输出

willie

调用方法

使用据点表示法可以调用类中定义的任何放方法

dog.sit()
dog.roll_over()

输出

Willie is now sitting.
Willie rolled_over!

创建多个实例

my_dog = Dog("willie", 4)
your_dog = ("lick", 3)

可以根据类创建多个实例。

属性的操作

指定属性默认值

类中的每个属性都必须有初始值，在方法init()内可以指定初始值，这样就无需在初始化类时提供形参。

class Dog():
    def __init__(self, name):
        self.name = name
        self.age = 1  # 指定小狗的age默认值为0

修改属性的值

可以通过三种方法修改属性的值：

  1. 直接修改属性值

     dog = Dog("willie")
 print("wollie 今年" + str(dog.age) + "岁")
 dog.age=1
 print("wollie长了1岁，今年" + str(dog.age) + "了")

    输出

     wollie 今年0岁
 wollie长了1岁，今年1了

  2. 通过方法设置属性值

     class Dog():

 def __init__(self, name):
     self.name = name
     self.age = 1

 def update_age(self, age):
     if self.age >= age:
         print("你狗狗的年龄不能倒着长")
     else:
         self.age = age

 dog = Dog("willie")
 print("wollie 今年" + str(dog.age) + "岁")
 dog.update_age(2)
 print("wollie长了1岁，今年" + str(dog.age) + "了")
 dog.update_age(1)

    输出

     wollie 今年1岁
 wollie长了1岁，今年2了
 你狗狗的年龄不能倒着长

    调用方法对狗狗的年龄进行修改，并且修改前对狗狗的年龄判断是否增加年龄，增加执行更行年龄，否则提醒不能减小狗狗的年龄。

  3. 通过方法递增递减属性值

     class Dog():

 def __init__(self, name):
     self.name = name
     self.age = 1

 def add_age(self, age):
     self.age += age

 dog = Dog("willwe")
 print(dog.age)
 dog.add_age(1)
 print(dog.age)

    输出

     1
 2

    这样每次调用add_age()方法并指定狗狗成长的年龄就可以实现对狗狗年龄属行的修改。

类的继承

一个类从一另一个类继承时，可以获得他的成员属性和成员方法。被继承的类称为父类（基类、超类），新类称为子类（派生类）。

class Person():

    def __init__(self, name, age):
        self.name = name
        self.age = age

    def say_name(self):
        print('我的名字是' + self.name)

class OldPerson(Person):

    def __init__(self, name, petname):
        super().__init__(name, petname)

old_person = OldPerson('xiaoming', '18')
print(old_person.name)
old_person.say_name()
  • 在子类名字的括号中写父类的名字既可以继承父类的非私有的所有属性和方法。并且子类可以调用父类的方法，访问父类的属性。
  • 在继承父类时，若父类init()需要参数，子类的init()需要协助父类赋值
