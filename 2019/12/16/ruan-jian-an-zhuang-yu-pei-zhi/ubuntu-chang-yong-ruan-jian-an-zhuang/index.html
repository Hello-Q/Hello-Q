Ubuntu常用软件安装

git

安装

sudo apt-get install git

vscode

安装

上官网下载安装包，直接点击安装即可。

设置中文

  • 按ctrl+shift+x打开商店，搜索Chinese (Simplified) Language Pack for Visual Studio Code 选择中文安装包安装。

  • 按ctrl+shift+p打开命令面板，之后输入 “config” 筛选可用命令列表，选择>display language，后选择zh-cn并重启scode。

解决终端乱码、字体太宽

点击文件→首选项→设置，搜索terminal.integrated.fontFamily，修改输入monospace

pycharm

安装

  • 上官网下载解压。
  • 进入/pycharm-2019.3/bin运行 ./pycharm.sh 按照指引配置

破解（ver=2019.3）

汉化

将汉化包放入lib文件夹内重启IDE

导入配置

选择文件-导入配置、到处配置进行操作

桌面图标

在/usr/share/applications/下创建pycarm.desktop，写入

[Desktop Entry]
Version=1.0
Type=Application
Name=PyCharm
Icon=/home/zhang/app/pycharm-2019.1.1/bin/pycharm.svg   #  图标路径
Exec="/home/zhang/app/pycharm-2019.1.1/bin/pycharm.sh" %f  #  执行文件路径
Comment=Python IDE for Professional Developers
Categories=Development;IDE;
Terminal=false
StartupWMClass=jetbrains-pycharm

python3.7

安装

  • 安装相关工具
apt-get install gcc make python-dev  libffi-dev libssl-dev  libxml2-dev libxslt-dev  libmysqlclient-dev libsqlite3-dev zlib1g-dev

缺少其他工具包，安装即可。

  • 下载安装包解压

    wget https://www.python.org/ftp/python/3.7.0/Python-3.7.0.tar.xz
tar -xvJf  Python-3.7.0.tar.xz

  • 编译安装

mkdir /usr/local/python3.7  # 创建编译安装目录
cd Python-3.7.0
./configure --with-ssl   --prefix=/usr/local/python3.7#注意加上ssl，不然会导致pip无法使用
make && make install
  • 创建软连接
sudo ln -s /usr/local/python3.7/bin/python3.7 /usr/local/bin/python3.7
sudo ln -s /usr/local/python3.7/bin/pip3.7 /usr/local/bin/pip3.7
  • 验证安装结果
zhang@zhang-pc:~$ python3.7 -V
Python 3.7.0
zhang@zhang-pc:~$ pip3.7 -V
pip 10.0.1 from /usr/local/python3.7/lib/python3.7/site-packages/pip (python 3.7)

Virtualenvwrapper

安装

sudo pip3 install virtualenvwrapper

可能的报错

ERROR: virtualenvwrapper could not find virtualenv in your path

解决方法
首先还是用 find / -name “virtualenv” 去找到位置，并建立软连接

zhang@zhang-pc:/usr/local/python3.7/bin$ sudo  find / -name "virtualenv" 
[sudo] zhang 的密码： 
/usr/local/python3.7/bin/virtualenv
find: ‘/run/user/1000/gvfs’: 权限不够
find: ‘/run/user/121/gvfs’: 权限不够
zhang@zhang-pc:/usr/local/python3.7/bin$ sudo ln -s /usr/local/python3.7/bin/virtualenv /usr/local/bin/virtualenv

配置

  • 编辑 ~/ .bashrc 文件

    export VIRTUALENVWRAPPER_PYTHON=/usr/local/python3.7/bin/python3.7  # 使用的python
export WORKON_HOME=~/.virtualenv # 存放虚拟环境的目录
source /usr/local/python3.7/bin/virtualenvwrapper.sh

  • 建立存放虚拟环境的目录

    mkdir ~/.virtualenv

  • 初始化

    source  ~/.bashrc

创建环境

mkvirtualenv python3

选择虚拟环境

workon python3

列出虚拟环境

workon

或者是

lsvirtualenv

停止环境

deactivate

删除环境

rmvirtualenv python3

设置github和码云ssh key

  • 创建相关文件夹

    mkdir ~/.ssh
cd ~/.ssh

  • 通过下面的命令，依次生成两个平台的key

    $ ssh-keygen -t rsa -C "xxxxxxx@qq.com" -f "github_id_rsa"
$ ssh-keygen -t rsa -C "xxxxxxx@qq.com" -f "gitee_id_rsa"

  • 把key复制到gitee和github

    cat github_id_rsa.pub

  • 创建config文件解决ssh冲突
    在.ssh文件夹下执行命令vi config
    文件中添加以下内容

    # gitee
Host gitee.com
HostName gitee.com
PreferredAuthentications publickey
IdentityFile ~/.ssh/gitee_id_rsa

github

Host github.com
HostName github.com
PreferredAuthentications publickey
IdentityFile ~/.ssh/github_id_rsa


- 测试  
执行

ssh -T git@github.com
成功则返回
Hi xxx! You’ve successfully authenticated, but GitHub does not provide shell access.

执行
ssh -T git@gitee.com
成功则返回
Hi xxx! You’ve successfully authenticated, but GITEE.COM does not provide shell access.
