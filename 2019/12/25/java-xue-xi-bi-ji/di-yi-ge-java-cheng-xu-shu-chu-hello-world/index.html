JAVA 程序编写流程

  1. 新建拓展名为.java的文件
  2. 编写相关代码
  3. 在文件的路径下启动命令行
  4. 使用javac对编写的文件进行编译，生成.class文件
  5. 使用java对生成的文件进行解释执行

第一个java程序

编写程序

文件名为Hello.java

public class Hello { // public class 是java的关键字，并且使用此关键字修饰的类名要与文件名相同
    public static void main(String[] args) { //main()方法是java程序执行的入口点
        System.out.println("Hello World！");  //向控制台输出信息
    }
} //java中{}要一一对应
//java大小写敏感，语句用半角分号结束

执行程序

  1. 在命令中执行javac hello 编译代码
  2. 接着执行java hello 运行代码

控制台会输出

Hello World
