java安装

  • 上官网下载对应安装包
  • 解压安装包，移动到/opt目录下
  • 添加环境

在~/.bashrc中添加以下代码

export JAVA_HOME=/opt/jdk1.8.0_231  ## 这里要注意目录要换成自己解压的jdk 目录
export JRE_HOME=${JAVA_HOME}/jre  
export CLASSPATH=.:${JAVA_HOME}/lib:${JRE_HOME}/lib  
export PATH=${JAVA_HOME}/bin:$PATH

java路径中不要出现中文

java JDK目录简介

目录 用途
bin 存放java的编译器、解释器（对应系统的可执行文件）
include 存放调用系统资源的借口文件
jre java运行环境文件
lib Java类库文件
src.zip JDK提供类的源代码

常用的概念

名称 概念
jdk java开发工具包
jre java运行环境信息
jvm java虚拟机
javac java语言编译器
java java语言解释器
