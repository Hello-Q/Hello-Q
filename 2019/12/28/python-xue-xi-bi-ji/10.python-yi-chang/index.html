异常

  • 错误分为错误和异常
  • 错误可以认为避免
  • 异常是只语法逻辑正确，出现的问题
  • python中，异常是一个类，可以处理使用

异常处理

python使用try/except语句捕获处理异常

try:
    pass
    # 正常执行语块
except expression:
    pass
    # 发生对应异常时执行语块
except (expression1, expression2, ...):
    pass
    # 发生异常若属于其中一种，执行语块
except expression as identifier:
    print(identifier)
    # 可以重命名错误，并使用他
    pass
else:
    pass
    # 没有异常发生时执行语块
finally:
    pass
    # 无论异常是否发生，都会执行的语块
  • 最少要有try、except两个语块，else和finally是可选的。
  • 多个except语块自上至下执行第一个错误匹配的且最多执行一个except语块。

抛出异常

使用raise语句可以抛出一个指定的异常。

raise语法

raise [Exception [, args [, traceback]]]
Exception：错误类型
args：可选，以元组的形式传递给异常的参数
traceback：可选，异常触发时新生成的一个用于异常-正常化的跟踪记录，多用于重新引发异常时

raise示例

x = input()
if x > 5:
    raise Exception('x 不能大于 5。x 的值为: {}'.format(x))

如果x大于五就会引发异常

自定义异常

通过继承Exceotion类可以自定义一个异常

NameError
class MyError(Exception):

    def __init__(self, value):
        self.value = value

    def __str__(self):
        return repr(self.value)


# 引发这个异常

raise MyError('自定义的异常')
