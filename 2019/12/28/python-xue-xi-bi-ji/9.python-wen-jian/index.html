open()函数

语法

open(file, mode=’r’, buffering=-1, encoding=None, errors=None, newline=None, closefd=True, opener=None)

参数说明:

file: 必需，文件路径（相对或者绝对路径）。
mode: 可选，文件打开模式
buffering: 设置缓冲
encoding: 一般使用utf8
errors: 报错级别
newline: 区分换行符
closefd: 传入的file参数类型

open mode模式可选值

模式 可读 可写 已存在文件写入方式 不存在写入处理方式 是否二进制
r ✓ x 读取 o x
rb ✓ x 读取 o ✓
r+ ✓ ✓ 从指针处覆盖原内容 报错 x
rb+ ✓ ✓ 从指针处覆盖原内容 报错 ✓
w x ✓ 全删重写 创建写入 x
wb x ✓ 全删重写 创建写入 ✓
w+ ✓ ✓ 全删重写 创建写入 x
wb+ ✓ ✓ 全删重写 创建写入 ✓
a x ✓ 追加 创建写入 x
ab x ✓ 追加 创建写入 ✓
a+ ✓ ✓ 追加 创建写入 x
ab+ ✓ ✓ 追加 创建写入 ✓
  • r开头主要是不能创建新文件
  • w开头主要是不能追加新内容
  • a开头则可以追加 内容
  • 二进制可以写一些图片等文件

with语句

with语句主要是可以自动判断文件作用域，在不需要时自动关闭文件，访问文件忘记关闭或过早关闭

with open(r'./test.txt', 'r') as f:
    contents = f.read()
    print(contents)

file对象属性方法

属性

属性 描述
file.close 检查文件是否关闭，返回bool值
file.mode 返回文件打开的模式
file.name 返回打开文件的名字

方法

方法 描述
file.close() 关闭文件
file.read(size) 读取文件内容，size设置读取字节数，为给定返回全部内容
file.readline(size) 按行读取，包括/n，size设置读取字节数，会读取到这个字节数所在行的结束位置。
file.readlines(size) 按行读取，包括/n，并讲每行当做一个元素组成一个列表，会读取到这个字节数所在行的结束位置。
flie.write(str) 写入内容，返回写入内容长度
file.writelines(sequence) 向文件写入一个序列字符串列表，如列表、元祖
file.seek(offset[, whence]) 调整指针位置，offset偏移量，whence可选，默认0：文件头，1：当前位置，2：文件尾
file.tell() 返回当前指针位置

with open(r'./test.txt', 'r+') as f:
