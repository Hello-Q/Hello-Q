Django安装

直接使用pip

pip install django

创建一个Django 项目

使用django-admin模块

# django-admin startproject <project_name>

django-admin startproject mysite

运行Django项目

使用Django manage 工具

python manage.py runserver

默认端口8000

  • 也可以指定端口
    python manage.py runserver 8001
  • 指定IP
    python manage.py 0:8001

在浏览器打开对应ip+端口可以祝贺页面代表成功

django 自带的web服务器支持热更新，但创建新文件等操作需要手动重启服务

创建一个应用

使用django manage工具创建应用

python manage.py start app polls

编写app视图

from django.http import HttpResponse


def index(request):
    return HttpResponse("Hello, world. You're at the polls index.")

为视图编写路由

在app的目录下创建urls.py

编写对应代码
*app下的urls.py

from django.urls import path

from . import views

urlpatterns = [
    path('', views.index, name='index'),
]

在项目目录urls.py中插入include()将路由指向app中的路由文件url.py

项目目录下的urls.py

from django.contrib import admin
from django.urls import include, path

urlpatterns = [
    path('polls/', include('polls.urls')),
    path('admin/', admin.site.urls),
]

path()参数

参数 描述
route url路径，匹配用户访问的路径
view url匹配后调用的视图
kwargs 任意个关键字参数可以作为一个字典传递给目标视图函数
name 为url命名，主要用在模板中
