安装

pip安装

适用与django2 + py3

  • pip安装
    pip install xadmin
  • github source
    pip install git+git://github.com/sshwsfc/xadmin.git
    github源码仓库
    下载源码后复制xadmin文件到相应位置（第三方app文件夹等）

配置

注册

在目录下新建adminx.py文件，并导入xadmin，使用xadmin.site.register对模型进行注册

import xadmin
from . import models

class <AppModelAdmin>(object):
    pass

xadmin.site.register(<model.ModelName>, <AppModelAdmin>)

模型编辑相关设置

属性 功能
list_display 指定列表页显示字段列表
search_fields 添加搜索字段
list_editable 指定字段可在列表页进行编辑，无需进入详情页
list_filter 列表页过滤字段
inlines 用于编辑一对多关联模型的数据。指定一个列表，元素为类，类的属性指定模型的关联模型，默认新增数量，风格（one，accordion，tab，stacked，table）

示例

class GoodsAdmin(object):
    """商品"""
    list_display = ["name", "click_num", "sold_num", "fav_num", "goods_num", "market_price",
                    "shop_price", "goods_brief", "goods_desc", "is_new", "is_hot", "add_time"]
    search_fields = ['name', ]
    list_editable = ["is_hot", ]
    list_filter = ["name", "click_num", "sold_num", "fav_num", "goods_num", "market_price", "shop_price", "is_new", "is_hot", "add_time", "category__name"]
    style_fields = {"goods_desc": "ueditor"}

    class GoodsImagesInline(object):
        model = models.GoodsImage
        exclude = ["add_time"]
        extra = 1
        style = 'tab'

    inlines = [GoodsImagesInline]

修改显示app名称

  • 修改app下apps.py的Config类，新增verbose_name 属性指定app名称，要求在settings注册app时，要使用.apps.Config的方式注册
  • 上述方式settings的注册方式为时，需要在app下的init.py文件中加入 default_app_config = “.apps.Config”指定app信息的类

筛选外键内容

如在表单中过滤外键的显示内容，参考以下代码

class GoodsBrandAdmin(object):
    """品牌"""
    list_display = ["name", "category", "image", "desc"]

    def get_context(self):
        context = super(GoodsBrandAdmin, self).get_context()
        if 'form' in context:
            context['form'].fields['category'].queryset = models.GoodsCategory.objects.filter(category_type=1)
            print(context['form'].fields['category'].queryset)
        return context
