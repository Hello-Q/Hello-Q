Python代码块

Python语言通过缩进来组织代码块，这是Python的强制要求。在Python编程中具有相同缩进的代码被自动视为一个代码块，无论进行了几个空格的缩进都是被允许的，只要缩进空格的数量统一。

以冒号开始，用缩进划分相同的作用域，这种结构成为代码块。

作用域指作用范围，同一个代码块拥有相同的作用域。

较为规范的情况下一般使用采用4个空格表示一个缩进。也可以使用一个制表符表示一个缩进（Tab键）。一般的IDE都拥有着把一个制表符视为4个空格的转换（记事本不具有此功能）。但一定要避免制表符和空格键的混用，以避免造成不必要的错误。

在代码结束时，多敲一次换行。使得代码层次结构更加清晰

分支结构

Python if 条件语句是通过一条或多条语句的执行结果（True或者False）来决定执行的代码块。

Python 编程中 if 语句用于控制程序的执行

单向分支

单项分支指只有一个条件判断，如果条件成立执行if代码块，否则程序忽略if代码块继续往下执行

if 判断条件：
    执行语句……

示例

a = 10
>>> if a < 12:
...        print('a小于12')
>>> print('if条件分支结束')
a小于12
if条件分支结束

双向分支

双向分支有一个条件条件判断，如果条件成立执行if代码块，否则执行else代码块

if 判断条件:
    执行语句...
else:
    执行语句...

示例

>>> a = 10
>>> if a <12:
...     print('a小于12')
... else:
...     print('a不小于12')
...
a小于12

多向分支

多向分支有多个判断条件，语句执行条件成立的第一个条件的语句块，不成立的条件，以及后面的语句块不执行。并且条件语句中可以使用逻辑运算符（and、or、not），语句形式如下

if 判断条件1:
    执行语句1……
elif 判断条件2:
    执行语句2……
elif 判断条件3:
    执行语句3……
else:
    执行语句4……

示例

soure = 90

if soure <= 100 and soure >= 90:
    print('成绩优秀')
elif soure == 90 and soure >=80:
    print('成绩良好')
elif soure < 80 and soure >=60:
    print('成绩及格')
elif soure < 60 and soure >=0:
    print('成绩不及格')
else:
    print('你特娘的是个成绩吗')

>>> 成绩优秀

Python中没有switch语句

巢状分支

巢状分支指单项分支、双向分支、多向分支嵌套使用
例如

# 巢状分支(单项分支,双向分支 多项分支,互相嵌套)

 youqian = True
 youfang = True
 youche = True
 youyanzhi = True
 youtili = True

#注意前面的缩进,来判断是否是同一块代码
 if youqian == True:
     if youfang == True:
         if youche == True:
             if youyanzhi == True:
                 if youtili == True:
                     print("恭喜你,牵手成功")
                 else:
                     print("恭喜你,成为1号替补")
             else:
                 print("老弟你等等,相约一九九八,就在明天")
 else:
     print("老弟,.快走吧你")

>>> 恭喜你,牵手成功

巢状分支最好不要超5层，太多容易造成逻辑混乱
