Python装饰器
Python 装饰器 语法糖
Python

装饰器可以在不改变原有类或函数的代码的前提下给原类或函数增加功能。

一个简单的装饰器

def outer(func):
    def inner():
        print('在运行老函数之前，我可以先做一下准备操作')
        f = func()
        print('在运行老函数之后，我可以先做一下善后操作')
        return f
    return inner


@outer
def old():
    print('这是一个老函数')
    return 1


print(old())

输出

在运行老函数之前，我可以先做一下准备操作
这是一个老函数
在运行老函数之后，我可以先做一下善后操作
1

实际上，装饰器是利用闭包函数来实现的，将老函数当做参数传入闭包函数中并在内函数中进行调用。

作用：可以在不改变现有函数的情况下，给函数添加一些功能（运行函数之前、之后都可以添加一些操作）

案例-使用装饰器统计函数运行时间

import time


def run_time(func):
    def inner():
        start = time.perf_counter()
        func()
        end = time.perf_counter()
        print(f'程序运行时间为{end-start}')
    return inner


@run_time
def func():
    for i in range(3):
        print(i, end=' ')
        time.sleep(1)


func()

输出

0 1 2 程序运行时间为3.0022267

多个装饰器装饰一个函数

def expand1(func):
    def inner():
        print('拓展1前')
        func()
        print('拓展1后')
    return inner


def expand2(func):
    def inner():
        print('拓展2前')
        func()
        print('拓展2后')
    return inner


@expand1
@expand2
def old():
    print('这是一个老函数')


old()

输出

拓展1前
拓展2前
这是一个老函数
拓展2后
拓展1后

当使用多个装饰器装饰一个函数时，程序会先将老函数从下到上依次放入装饰器进行装饰

old先放入@expand2进行装饰并返回inner函数，然后再将返回的inner函数放入@expand1进行装饰，然后开始执行函数。

装饰带有参数的函数

def outer(func):
    def inner(name):
        print(f'you name is {name}')
        func(name)
    return inner


@outer
def old(name):
    print(f'my name is {name}')


old('xiaohong')

输出

you name is xiaohong
my name is xiaohong

装饰器返回的是实际是闭包函数中的内函数，所以要把函数的参数通过内函数进行传入并在内函数调用老函数时传入。

装饰多参数的函数

def outer(func):
    def inner(name, *args, **kwargs):
        print(f'打电话给{name},约她出来玩')
        func(name, *args, **kwargs)
        print('玩完各回各家，各找各妈了。')
    return inner


@outer
def happy_day(name, *args, **kwargs):
    """
    约朋友开心玩一天
    :param name: 约的谁
    :param args: 吃了啥
    :param kwargs: 玩了啥
    :return:
    """

    print(f'成功约到{name}一起玩耍')
    print(f'和她一起吃了{args}')
    print(f'去{kwargs["play"]}玩了，还看了一部电影叫{kwargs["movie"]}')


happy_day('小白', '冰激凌', '武汉鸭脖', movie='坦坦尼克号', play='游乐场')

输出

打电话给小白,约她出来玩
成功约到小白一起玩耍
和她一起吃了('冰激凌', '武汉鸭脖')
去游乐场玩了，还看了一部电影叫坦坦尼克号
玩完各回各家，各找各妈了。

类似的，多参数装饰器要求把所有参数也在装饰器内函数中传入。

带有参数的装饰器

def whether(weather):  # 再加一层外壳接受参数
    def outer(func):
        def inner1(name, *args, **kwargs):
            print(f'{name}说今天天气不错')
            func(name, *args, **kwargs)
            print('玩完各回各家，各找各妈了。')

        def inner2(name, *args, **kwargs):
            print('算了吧，今天天气不好')
        if weather == '晴朗':  # 根据参数进行具体业务
            return inner1
        else:
            return inner2
    return outer


@whether('晴朗')
def happy_day(name, *args, **kwargs):
    """
    约朋友开心玩一天
    :param name: 约的谁
    :param args: 吃了啥
    :param kwargs: 玩了啥
    :return:
    """

    print(f'成功约到{name}一起玩耍')
    print(f'和她一起吃了{args}')
    print(f'去{kwargs["play"]}玩了，还看了一部电影叫{kwargs["movie"]}')


happy_day('小白', '冰激凌', '武汉鸭脖', movie='坦坦尼克号', play='游乐场')

带有参数的装饰器，需要加一层外壳来接受装饰器的变量，这样装饰器可以根据传入的参数进行一些操作，并逐层返回函数。

使用类装饰器装饰函数

class Outer:

    def __call__(self, func):
        self.func = func  # 将传入的func参数赋到类方法
        return self.inner

    def inner(self, name):
        print('今天天气不错')
        self.func(name)


@Outer()
def happy_day(name):
    print(f'和{name}出去玩咯')


happy_day('小白')

使用类装饰一个函数，其实相当于把类当做函数调用，当一个类当做函数调用时，__call__魔术方法会自动调用

  1. @Outer()会把被装饰的函数当做参数传入类调用；
  2. __call__接受func参数，并把它赋给类方法，并返回inner方法；
  3. inner方法接受name参数进行处理并调用happy_day函数；

传递流程：

Outer(happy_day) –> __call__(self, happy_day) –> inner(self, name) –> self.func(name) ==> hapay_day(name)

使用类方法装饰一个函数

class Outer:

    def inner(func):
        Outer.func = func
        return Outer.new_inner

    @staticmethod
    def new_inner(name):
        print('今天天气不错')
        Outer.func(name)


@Outer.inner
def happy_day(name):
    print(f'和{name}出去玩咯')


happy_day('小白')

使用一个类的方法装饰一个函数， 将这个函数传递给类函数（inner），类函数将函数储存为类方法，并返回另一个类方法(new_inner)，在新的类函数中调用储存的类方法（Outer.func)

装饰流程：

@Outer.inner(happy_day) –> inner(happy_day) –> new_inner(name) –> Outer.func(name) ==> happy_day(name)

使用函数装饰器装饰类

def expand(cls):
    def func2(self):
        print('这是拓展的function2')
    cls.func2 = func2
    cls.name = 'Demo类'
    return cls

@expand
class Demo:

    def func1(self):
        print('这是function1')


demo = Demo()
demo.func1()
print(demo.name)
demo.func2()

使用一个函数装饰一个类，就是将这个类作为参数传入装饰器函数中，为其增加属性并返回。

使用类装饰器装饰类

class Expand:
   def __call__(self, cls):
      self.cls = cls
      return self.new_func

   def new_func(self):
      self.cls.name = 'Demo类'
      self.cls.func2 = self.func2
      return self.cls()

   def func2(self):
      print('这是新增的function2')


@Expand()
class Demo:

   def func1(self):
      print('这是function1')


demo = Demo()
demo.func1()
print(demo.name)
demo.func2()

使用类装饰器装饰类，是将类放入类装饰器中，通过调用__call__函数，将要装饰的类（Demo）传入，__call__函数将传入的类赋值给自己的属性，并返回另一个函数（相当于闭包函数中的内函数），这个函数为要装饰的类增加新的属性，并返回类的调用对象。
